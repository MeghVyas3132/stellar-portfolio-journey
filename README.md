## ğŸš€ Getting Started

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd stellar-portfolio-journey

# Install dependencies
npm install
# or
bun install
```

```sh

bun run dev

# or

```
